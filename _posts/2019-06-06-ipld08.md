# Specification: DAG-JSON

DAG-JSON은 전체 [IPLD 데이터 모델]()을 지원한다.

## Format

### Simple Types

바이너리를 제외한 모든 간단한 유형은 기본적으로 JSON에서 지원된다.

JSON은 Big Integers를 지원한다. 즉, `dag-json`의 JS 구현은 기본 JSON 파서 및 직렬기를 사용할 수 없다는 것을 의미한다.

#### Binary Type

```javascript
{"/": { "base64": String }}
```

### Link Type

```javascript
{"/": String /* base encoded CID */}
```
