# Concept: Block

IPLD 블록은 해당 CID에 대한 CID 및 이진 데이터 값이다.

short version:

```tex
+-----+--------------------------------+
| CID | Data                           |
+-----+--------------------------------+
```

long version:

```tex
+-----------------------------------+------------------+
| CID                               | Binary Data      |
| +------------------------------+  |                  |
| |Codec                         |  |                  |
| +------------------------------+  |                  |
| |Multihash                     |  |                  |
| | +----------+---------------+ |  |                  |
| | |Hash Type | Hash Value    | |  |                  |
| | +----------+---------------+ |  |                  |
| |                              |  |                  |
| +------------------------------+  |                  |
|                                   |                  |
+-----------------------------------+------------------+
```
